# 三俣山荘グループ WEBサイト デザインコンセプト

**作成日**: 2025年11月19日 | **バージョン**: 1.0

---

## 🎨 デザインの基本思想

### 「詩情と実用性の調和 - ストーリーテリングで伝える山の世界」

北アルプス最奥、黒部源流に位置する三俣山荘グループ。その本質は「山の広さと深さへの入口」です。

このWEBサイトのデザインは、**山の詩情的な美しさと、登山者が必要とする実用的な情報**を、5:5のバランスで表現します。

---

## 💫 コアコンセプト

### 1. ストーリーテリング形式

**雑誌のようなエディトリアルデザイン**

- 大きな写真が物語る山の世界
- 左右非対称のレイアウトで視覚的なリズムを生む
- スクロールすることで深まる没入感
- 各山荘の個性を、写真とテキストの配置で表現

### 2. 和の美意識

**日本の山小屋としてのアイデンティティ**

- 和紙のテクスチャを背景に使用
- 繊細な装飾線とアクセント
- 余白を活かした上品なレイアウト
- 日本語タイポグラフィの美しさを追求

### 3. 没入感のある体験

**訪れる前に、山を感じられる**

- パララックススクロールによる奥行き
- 大きなヒーローイメージでの歓迎
- スムーズなアニメーションとトランジション
- 写真の質と配置による情緒的な表現

## 🖼️ ビジュアルデザイン

### 写真の使い方

#### 1. ヒーローセクション

**パララックススライドショー**

- 8枚の厳選された写真（デスクトップ）
- 4枚に最適化（モバイル）
- 7秒間隔のフェード切り替え
- 黒のグラデーションオーバーレイで文字の可読性を確保

**写真の選定基準:**
- 北アルプスの雄大さが伝わるもの
- 四季の変化が感じられるもの
- 山荘の雰囲気が伝わるもの
- 人の温かみが感じられるもの

#### 2. 山荘紹介セクション

**ストーリーテリング形式のレイアウト**

```
[タイトル・リード文（右or左寄せ）]
    ↓
[縦写真（中サイズ）- 左or右]
    ↓
[横写真（大サイズ）- 反対側]
    ↓
[全幅写真 or 縦写真]
```

**写真のアスペクト比:**
- 縦写真: 3:4（ポートレート）
- 横写真: 3:2（ランドスケープ）
- 全幅写真: 4:3

**インタラクション:**
- マウス追従の3D傾きエフェクト（デスクトップ）
- ホバー時の微細なスケールアップ（1.015倍）
- スクロール連動のフェードイン

### レイアウトシステム

#### グリッドシステム

**最大幅:**
```
max-w-7xl (1280px) - 標準コンテナ
max-w-5xl (1024px) - テキストコンテンツ
max-w-3xl (768px) - リード文・説明
```

**カラム構成:**
- デスクトップ: 3カラムグリッド（お知らせなど）
- タブレット: 2カラム
- モバイル: 1カラム（横スクロールカルーセル）

#### 余白の哲学

**「間」を大切にする和の美意識**

```
セクション間: py-16 md:py-32 lg:py-40
コンテナ: px-6 md:px-12 lg:px-20
要素間: 適切な「間」で呼吸するレイアウト
```

余白は単なる空間ではなく、コンテンツに「呼吸」を与える重要な要素。

## ✍️ タイポグラフィ

### フォント選定

**Noto Serif JP - 優雅さと可読性**

```
用途: 見出し・本文
ウェイト: Light (300) - メイン
特徴:
- 和文に最適化された明朝体

- 長文でも読みやすい
```

**Noto Sans JP - 機能性**

```
用途: UI・ラベル・補助テキスト
ウェイト: Light (300), Regular (400)
特徴:
- クリーンで現代的
- 小さいサイズでも可読性が高い
```

### 文字サイズ階層

**大胆で美しいタイポグラフィ**

```
Hero Title:     48px → 96px（3xl → 6xl）
Section Title:  32px → 64px（2xl → 4xl）
Body:           14px → 18px（sm → lg）
Caption:        12px → 14px（xs → sm）
```

**レスポンシブに拡大縮小し、デバイスに最適なサイズを提供**

### 字間・行間

**読みやすさと美しさのバランス**

```css
行間（leading）: 1.6 - 1.8
  → ゆったりとした読みやすい行間

字間（tracking）:
  和文: 0.04em - 0.08em
    → 和文の美しさを引き立てる繊細な字間

  英文・ラベル: 0.2em
    → 大文字の視認性を高める広めの字間
```

  → 詩的すぎて意味が不明確

## ✨ インタラクション・アニメーション

### アニメーション原則

**「控えめだが、心地よい」**

1. **意味のあるアニメーション**
   - ユーザーの行動に対するフィードバック
   - 視線誘導
   - 階層構造の理解を助ける

2. **パフォーマンス優先**
   - モバイルでは軽量化
   - 60fps維持
   - prefers-reduced-motion対応

3. **自然な動き**
   - イージング: `cubic-bezier(0.22, 1, 0.36, 1)`
   - 継続時間: 0.3 - 1.2秒
   - 過度に速くも遅くもない

### 主要なインタラクション

#### 1. パララックス効果

```
用途: ヒーローセクション
効果: 背景が前景よりゆっくり動く
目的: 奥行き感、没入感の演出

デスクトップ: 有効
モバイル: 無効（パフォーマンス優先）
```

#### 2. ホバーエフェクト

```
写真: scale(1.015) - 微細な拡大
ボタン: scale(1.02-1.05) - 明確なフィードバック
カード: 影の変化 - 浮き上がる感覚
```

#### 3. スクロールアニメーション

```
フェードイン: opacity 0 → 1 + y移動
遅延: 要素ごとに0.1秒ずつ遅延
目的: ページに生命を吹き込む
```

#### 4. ページ遷移

```
Framer Motion のページトランジション
スムーズで途切れない体験
```

## 🌈 カラーシステム

### 基本カラー

```
背景: #fcf6e3
  → 和紙風の温かみのあるオフホワイト
  → 長時間見ても目が疲れない

テキスト: #0a0a0a
  → 純黒ではない、柔らかい黒
  → 高いコントラストで読みやすい

グレー: #525252, #737373, #d1d5db
  → 階層を表現する3段階のグレー
```

### 配色の考え方

**自然から学ぶ**

- 和紙の温かみ → 背景色
- 黒部源流の深い緑 → アクセント（三俣山荘）
- 稜線の青空 → アクセント（水晶小屋）
- 紅葉の赤茶 → アクセント（湯俣山荘）

**色の使用ルール:**
- 基本は白・黒・グレーの無彩色
- 各山荘のアイデンティティカラーをアクセントに
- 派手すぎず、品よく

## 📱 レスポンシブデザイン

### モバイルファースト

**設計思想:**
1. モバイルで完璧に機能する
2. タブレット・PCで体験を拡張する
3. デバイスごとに最適化する

### ブレークポイント

```
sm:  640px  - 小型タブレット
md:  768px  - タブレット
lg:  1024px - 小型デスクトップ
xl:  1280px - デスクトップ
2xl: 1536px - 大型ディスプレイ
```

### デバイス別の最適化

#### モバイル（〜767px）

**パフォーマンス優先:**
- 画像枚数削減（8枚→4枚）
- パララックス無効化
- Ken Burnsエフェクト無効化
- backdrop-blur削減

**UI最適化:**
- 横スクロールカルーセル
- タッチ操作に最適化
- 大きなタップターゲット
- シンプルなナビゲーション

#### タブレット（768px〜1023px）

**バランス:**
- 一部のアニメーション有効化
- 2カラムレイアウト
- タッチとマウス両対応

#### デスクトップ（1024px〜）

**フル体験:**
- すべてのアニメーション
- ホバーエフェクト
- パララックス
- 3Dエフェクト
