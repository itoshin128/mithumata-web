## エグゼクティブサマリー

### 美しく、わかりやすく、心に残るサイトへ

三俣山荘グループのWEBサイトは、詩的で情緒的な表現により山の世界観を伝えることには成功していますが、ユーザーが実際に必要とする情報へのアクセスが困難という課題を抱えています。

本プロジェクトは、「**山の広さと深さへの入口**」というコンセプトのもと、山小屋の本質的価値である「山との深いかかわり」と「道直し」などの活動を効果的に発信し、共感を持って訪れる登山者を増やすことを目指します。

### プロジェクト概要

| 項目 | 内容 |
|------|------|
| 現状の課題 | 詩的な表現に偏重し、予約方法・アクセス情報・宿泊案内などの実用情報へのアクセスが困難 |
| リニューアルの方向性 | **「詩情:実用 = 5:5」** のバランス。山小屋としての詩情を保ちながら、必要な情報にストレスなくアクセスできる構造を実現 |
| デザインアプローチ | **ストーリーテリング形式**による没入感のある体験と、実用情報への明確な導線を両立 |
| 予算 | フェーズ1: 100-150万円 / フェーズ2: +50-100万円(オプション) |
| 公開予定 | 2026年3月上旬(一般予約開始前) |

## 2. 現状サイトの詳細分析

### 2-1. 現状サイトの構造

| カテゴリ | 主要ページ | 評価 |
|----------|------------|------|
| トップページ | `/` | 詩的表現に偏重、情報への導線が不明瞭 |
| 山荘情報 | `/mitsumata/`, `/suisho/`, `/yumata/` | 宿泊情報と魅力ページが分離、情報が散在 |
| ブログ | `/blog/` | 更新頻度が高く良好、検索性に課題 |
| お知らせ | `/news/` | ブログとの違いが不明確 |
| 伊藤新道 | `/itoshindo/` | 独立ページとして存在、情報は豊富 |

### 2-2. 具体的な課題抽出

#### 課題① トップページの情報設計

**現状:**
> 「北アルプス最奥地にして、2500m以上の標高にありながら、ゆったりとした山容に、原始と変わらぬ生態系の奥行きが息づいている黒部源流。1945年に先代伊藤正一が初めて足を踏み入れた当時、職猟師と物の怪だけが手中に収めていた…」

**問題点:**
- 雰囲気は伝わるが、実用情報へのアクセスが困難
- 初訪問ユーザーが「何をすればいいか」が不明確
- 予約ボタンや重要情報への導線が弱い

#### 課題② 予約システムへの導線

**問題:** 「予約サイトがどこにあるかわからない」という問い合わせが多発

**原因:**
- 予約ボタンが目立たない
- 予約開始日時の案内が埋もれている

#### 課題③ スマホでの閲覧性

**問題:** スクロールが多すぎる、文章が凝縮されて見づらい

**具体的課題:**
- 各写真ごとにフルスクリーン表示される構造
- 文字サイズと行間の最適化が不十分
- タッチ操作に最適化されたUIでない

## 4. プロジェクトビジョン

### 「山の広さと深さへの入口」
**お客さんと山の接点のきっかけとなる場所**

三俣山荘グループは、単なる宿泊施設ではありません。訪れる人々が「山の広さと深さ」と出会い、新たな山の楽しみ方を発見する入口となる場所です。

このWEBサイトは、その「接点のきっかけ」をデジタル空間で実現します。

### 4-1. 核となる2つのコンセプト

#### ① 泊まるだけではない山小屋

山小屋の公共的役割、地域保全、道直し活動など、宿泊以外の価値を可視化します。

#### ② 山小屋の「今」と「考え」を継続的に発信

コラム・連載を通じて、山小屋が今考えていることを伝え、訪れる人々との継続的なつながりを築きます。

### 4-2. デザインコンセプト

**「詩情と実用性の調和 - ストーリーテリングで伝える山の世界」**

- **没入感のあるビジュアル体験**：大きな写真とパララックス効果で山の世界に引き込む
- **和の美意識**：和紙テクスチャ、木の影、日本語タイポグラフィによる詩的な表現
- **明確な情報設計**：美しさの中に実用情報への明確な導線を確保
- **季節の演出**：営業期間(7-11月)に合わせた季節テーマの自動切り替え
- **詩情:実用性 = 5:5** のバランス
- みかの写真を効果的に活用し、山荘の「今」を視覚的に伝える

### 4-3. 参考サイト

本プロジェクトは、以下のサイトのデザイントーンと情報設計を参考にします:

#### ① sankara hotel & spa 屋久島
https://www.sankarahotel-spa.com/

- シンプルで洗練されたレイアウト
- 余白の効果的な使い方
- 大きな写真と最小限のテキスト
- ナチュラルトーンの配色

#### ② 甲羽湯
https://www.kohanyu.jp/

- 和の要素を取り入れた温かみ
- 情報の見せ方(階層構造)
- 温泉・食事の魅力的な表現
- モバイル最適化

### 7-2. トップページの詳細構成（実装版）

#### 1. ヒーローセクション（パララックス）

**デスクトップ:**
- 全画面スライドショー（8枚の写真、7秒間隔）
- Ken Burnsエフェクト（緩やかなズーム）
- パララックススクロール効果

**モバイル:**
- パフォーマンス最適化（4枚の写真に削減）
- パララックス無効化
- Ken Burnsエフェクト無効化

**コピー:**
```
メインタイトル: 北アルプス黒部源流
英語サブタイトル: Northern Alps Kurobe Genryu

キャッチコピー: 北アルプス最奥、黒部源流の三つの山荘
リード文: 原始と変わらぬ生態系が息づく場所。
```

**「わたしたちについて」セクション:**
- 左: 大きなキャッチコピー「山の広さと深さへの、入口」
- 右: 説明文（三俣山荘グループの理念）
- CTA: 「三俣山荘グループについて」ボタン

#### 2. 三俣山荘グループ紹介（ストーリーテリング形式）

**デザインアプローチ:**
- 雑誌のようなエディトリアルデザイン
- 大きな写真（縦構図・横構図を交互に配置）
- 左右非対称のレイアウトで視覚的なリズムを作る
- 各山荘ごとに3枚の写真を表示

**各山荘の構成:**
```
[山荘名 + リード文（右寄せ or 左寄せ）]
↓
[縦写真（中サイズ）- 左 or 右寄せ]
↓
[横写真（大サイズ）- 右 or 左寄せ]

↓
[「〇〇山荘の詳細」ボタン]

```

**山荘の順番:**
1. 湯俣山荘（右寄せ）

- 3D傾きエフェクト（マウス追従）

- スムーズなフェードイン（スクロール連動）
