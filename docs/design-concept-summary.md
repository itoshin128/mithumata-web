# 三俣山荘グループ WEBサイト デザインコンセプト（サマリー版）

**作成日**: 2025年11月19日 | **バージョン**: 1.0

---

## 🎨 コアコンセプト

### 「詩情と実用性の調和 - ストーリーテリングで伝える山の世界」

**詩情:実用 = 5:5** のバランスで、山の美しさと必要な情報を両立させる。

---

## 💫 3つの柱

### 1. ストーリーテリング形式
- 雑誌のようなエディトリアルデザイン
- 大きな写真が物語る山の世界
- 左右非対称のレイアウトで視覚的リズム

### 2. 上品で洗練された美意識
- 繊細なテクスチャと装飾
- 余白を贅沢に使った上品なレイアウト
- 洗練されたタイポグラフィ

### 3. 没入感のある体験
- パララックススクロール
- スムーズなアニメーション
- 写真の質と配置による情緒的表現

---

## 🎯 デザイン原則

| Before | After |
|--------|-------|
| 詩的すぎて情報が見つからない | 詩情と実用性のバランス |
| スクロールが多すぎる | ストーリー性のある適切なスクロール |
| モバイルで見づらい | モバイルファースト |
| 予約導線が不明確 | 常に見える予約ボタン |

---

## 🖼️ ビジュアル

### 写真の使い方

**ヒーローセクション:**
- 8枚のスライドショー（デスクトップ）
- パララックス効果で奥行き

**山荘紹介:**
```
[タイトル・リード文]
    ↓
[縦写真（中）]
    ↓
[横写真（大）]
    ↓
[全幅写真]
```

### レイアウト

**グリッド:**
- デスクトップ: 3カラム
- タブレット: 2カラム
- モバイル: 1カラム（横スクロール）

**余白:**
```
セクション間: py-16 → py-40
コンテナ: px-6 → px-20
適切な「間」で呼吸するレイアウト
```

## ✍️ タイポグラフィ

### フォント

**Noto Serif JP（メイン）**
- 見出し・本文
- Light (300) - 優雅で可読性が高い

**Noto Sans JP（サブ）**
- UI・ラベル
- クリーンで現代的

### サイズ階層

```
Hero:     48px → 96px
Section:  32px → 64px
Body:     14px → 18px
```

### 字間・行間

```
行間: 1.6 - 1.8
字間: 0.04em - 0.08em（和文）
     0.2em（英文・ラベル）
```

## 🎭 トーン＆マナー

### ビジュアル

| 要素 | 方向性 |
|------|--------|
| 写真 | 自然な色味、過度な加工を避ける |
| 配色 | 和紙の温かみ、山の色彩 |
| 余白 | 贅沢に使い、上品さを演出 |
| 装飾 | 最小限、シンプルな線と点 |

### 言葉

```
○ 「北アルプス最奥、黒部源流の三つの山荘」
  → 簡潔で印象的

× 「職猟師と物の怪だけが…」
  → 詩的すぎて意味不明
```

## ✨ インタラクション

### アニメーション原則

**「控えめだが、心地よい」**

1. 意味のあるアニメーション
2. パフォーマンス優先（モバイル軽量化）
3. 自然な動き（0.3-1.2秒）

### 主要エフェクト

| エフェクト | 用途 | 設定 |
|-----------|------|------|
| パララックス | ヒーロー | デスクトップのみ |
| ホバー | 写真・ボタン | scale(1.015-1.05) |
| フェードイン | スクロール | 0.1秒ずつ遅延 |

## 🎪 主要UIコンポーネント

### 予約ボタン

**デスクトップ:** 右側中央固定、縦書き「予約する」
**モバイル:** 右下固定、丸いボタン「予約」

### メニュー

右上の半透明円形ボタン → 右からスライドイン

### カード

丸角、柔らかい影、ホバーで浮き上がる

---

## 📱 レスポンシブ

### モバイルファースト

**モバイル（〜767px）:**
- パフォーマンス優先（画像削減、アニメーション軽量化）
- 横スクロールカルーセル
- 大きなタップターゲット

**タブレット（768px〜1023px）:**
- 2カラムレイアウト
- バランスの取れた体験

**デスクトップ（1024px〜）:**
- フル体験（すべてのアニメーション・エフェクト）
- ホバーエフェクト

## 📐 ガイドライン

### Do's ✓

- 大きな写真を効果的に使う
- 余白を贅沢に使う
- シンプルで明快な言葉
- アクセシビリティを考慮

### Don'ts ✗

- 小さな写真を詰め込む
- 詩的すぎて意味不明な表現
- 派手すぎる装飾
- 重いアニメーション

---

## 🎯 成功指標

1. **UX** - 予約ボタンの発見率、情報アクセス時間
2. **ビジュアル** - 山の世界観が伝わるか
3. **パフォーマンス** - 読み込み速度、スムーズさ
4. **感情** - 「行きたい」と思わせるか
