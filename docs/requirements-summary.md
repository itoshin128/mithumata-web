# 三俣山荘グループ WEBサイトリニューアル 要件定義書（サマリー版）

**作成日**: 2025年11月7日 | **更新日**: 2025年11月19日 | **バージョン**: 2.0

---

## 🎯 プロジェクトの目的

### 「山の広さと深さへの入口」- 詩情と実用性の調和

**現状の課題**: 詩的表現に偏重し、予約・アクセス情報など実用情報へのアクセスが困難

**目指す姿**: 山小屋の詩情を保ちながら、必要な情報にストレスなくアクセスできる「**詩情:実用 = 5:5**」のバランス

| 項目 | 内容 |
|------|------|
| 公開予定 | 2026年3月上旬（一般予約開始前） |
| 予算 | フェーズ1: 100-150万円 / フェーズ2: +50-100万円 |
| 対象 | 三俣山荘・水晶小屋・湯俣山荘の3施設統合サイト |

## 🎨 デザインコンセプト

### ストーリーテリング形式で伝える山の世界

**トップページの構成:**

1. **ヒーローセクション**（パララックス）
   - 全画面スライドショー + 大きなキャッチコピー
   - 「北アルプス最奥、黒部源流の三つの山荘」
   - 固定予約ボタン（右側縦書き / モバイル右下）

2. **三俣山荘グループ紹介**（エディトリアルデザイン）
   - 雑誌のような大きな写真レイアウト
   - 左右非対称の配置で視覚的リズムを作る
   - 各山荘3枚の写真 + リード文

3. **お知らせ**
   - デスクトップ: 3カラムグリッド
   - モバイル: 横スクロールカルーセル

4. **FAQ**（アコーディオン）
   - 予約・食事・設備・アクセスの4カテゴリ

### Before → After

| Before（現状） | After（リニューアル後） |
|---------------|----------------------|
| 詩的な長文、何をするサイトか不明確 | 一目で分かる山小屋サイト、明確な予約導線 |
| 縦スクロールが多い | ストーリーテリング形式で没入感 |
| 情報が散在 | カテゴリ別に整理、素早いアクセス |

## ⚙️ 主要機能要件

### 1. 予約導線
- ✅ デスクトップ: 右側中央に縦書き固定ボタン
- ✅ モバイル: 右下にフローティングボタン
- 外部予約システムへの直接リンク

### 2. レスポンシブデザイン
- ✅ スマホ・タブレット・PC最適表示
- ✅ モバイル: パフォーマンス最適化（画像削減、アニメーション制御）
- ✅ 横スクロール対応（お知らせカルーセル）

### 3. コンテンツ管理
- お知らせ・コラム機能（一覧・個別記事・カテゴリ分類）
- 社内スタッフが簡易更新可能（CMS検討）

### 4. ナビゲーション

**ヘッダーメニュー（推奨）:**
- お知らせ (/news) ← 新規追加
- コラム (/column) ← /blog から変更
- 山荘紹介 (/lodges)
- アクセス (/access)
- FAQ (/faq)
- 予約する (/reservations)
